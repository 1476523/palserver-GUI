# palserver GUI

### 1.0.0 正在開發中! 即將帶來更多實用的功能! 

- 伺服器多開 (支援平行運行)
- 伺服器複製、匯出、遠端連線等功能。
- 模組、差件、日誌環境獨立。
- 更完善的效能監控、資源管理及玩家管理。
- 自動重啟、崩潰重啟。
- 更完善的自動備份系統 (上限設置)。
- 全新 UIUX 升級 & 還有更多。


因為早期程式寫得較隨意、將程式碼重構花了較多時間，請大家再給我一些時間。

---

全圖形化介面，一鍵式帕魯 (Palworld) 伺服器建立、管理工具。

- 易於上手，只需點擊首頁`右下方按鈕`即可啟動伺服器。
- 提供完整的 `GUI 介面`調整世界設定、伺服器設定等。
- 支持`多存檔`及`模組`管理，方便設定不同玩法和好友同樂！
